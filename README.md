# Planner

## Description

This planner was designed to help people schedule their day by entering the event on the time slot
the color scheme per time slot will change over time to highlight the event time approaching. the current date is also displayed. 


## Installation

n/a

## Usage

Simple enter in an event and press the save button to retain the information
